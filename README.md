## 📖 Overview

PulseGuard AI is an intelligent healthcare platform designed to assist in the early detection of cardiovascular abnormalities through heart sound analysis.

The system processes phonocardiogram (PCG) recordings, extracts meaningful acoustic features, and leverages machine learning models to provide AI-assisted cardiac pre-screening results.

> ⚠️ This project is intended for educational, research, and pre-screening purposes only and does not replace professional medical diagnosis.

## 🚀 Installation

### Clone Repository

```bash
git clone https://github.com/YOUR_USERNAME/PulseGuard_v3.git
cd PulseGuard_v3
```

### Install Dependencies

Frontend

```bash
cd client
npm install
```

Backend

```bash
cd ../server
npm install
```

Python Environment

```bash
pip install -r requirements.txt
```

---

## 🐳 Docker Setup

Build and Run

```bash
docker-compose up --build
```

Run in Background

```bash
docker-compose up -d
```

Stop Containers

```bash
docker-compose down
```
